# node-server-kit

### Starter kit setup for node server with:

* MondoDB
* Moongoose
* Express
* ExpressValidator
* Socket.io 
* Nodemon
* Bcrypt
* Heroku
* Csurf

### To run this server:
* Copy or download repo
* cd server
* npm install
* nodemon.json file and add env configuration for mondoDB access, setup bellow
* nodemon app.js

* {
  * "env": {
   * "MONGO_USER": "XXXXXXXX", 
   * "MONGO_PASSWORD": "XXXXXXXXXXX",
   * "MONGO_DB": "XXXXXXXXXXX"
  * }
* }

### Enjoy :)
